# Chat-bot Installation Guide

## prerequisites

[NODE](https://nodejs.org/en) should be installed

## Installation

Use the package manager npm to install foobar.

```bash
npm init -y
```
```bash
npm install
```

## Run the Server

```bash
nodemon server.js
```

or

```bash
node server.js
```

## Open in browser

```bash
localhost:3000
```
